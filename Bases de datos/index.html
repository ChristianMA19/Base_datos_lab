Universidad del norte - proyecto bases de datos

En este proyecto usamos una base de datos de google playstore que conseguimos en esta pagina https://www.kaggle.com/datasets/gauthamp10/google-playstore-apps, Escogimos esta base de datos porque queriamos ser diferentes a los demas y no basarnos en la de covid. Esta base de datos utilizada contaba con más de 2 millones de datos los cuales redujimos a 147.209 datos.

La cantidad de datos unicos que nos brindo esta base de datos nos dejo crear una mayor variedad de graficos con mayor relatividad y mas varianza. Tambien nos ayudo con su amplio rango de atributos a crear diferentes cantidades en cada grafico, asi generando un resultado mas concreto y preferible al usarlo para analisar estos datos. Esto se realizo gracias a el uso de canvas.js que nos permitio realizar los graficos con facilidad y presicion, y sin ningun problema sobre la cantidad de datos en uso, la varianza de ellos o el uso aplicado.
